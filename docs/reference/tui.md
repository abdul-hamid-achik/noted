# TUI keybindings

Run `noted` with no arguments to open the interactive terminal UI.

## Global

| Key | Action |
|-----|--------|
| `1`–`9` | Switch view |
| `Tab` | Toggle sidebar |
| `Ctrl+K` | Command palette |
| `Ctrl+O` | Quick switcher |
| `?` | Help overlay |
| `q` / `Ctrl+C` | Quit |

## Views

| Key | View | What it does |
|-----|------|--------------|
| `1` | Notes | Browse notes; open in the editor |
| `2` | Search | Live full-content search |
| `3` | Tags | Filter notes by tag |
| `4` | Folders | Filter notes by folder |
| `5` | Tasks | Extracted checkboxes; `space` toggles done |
| `6` | Daily | Open/create today's daily note |
| `7` | Templates | New note from a template |
| `8` | Dashboard | Knowledge-base stat cards |
| `9` | Settings | Theme / paths / keybindings |

## Notes view

| Key | Action |
|-----|--------|
| `↑`/`↓` or `j`/`k` | Move |
| `/` | Filter |
| `n` | New note |
| `Enter` / click | Open note |
| `d` | Delete (press twice) |
| `Esc` | Clear tag/folder filter |

## Editor

| Key | Action |
|-----|--------|
| `[[` | Link autocomplete |
| `Ctrl+S` | Save |
| `Ctrl+L` | Follow a `[[wikilink]]` |
| `Ctrl+B` | Backlinks |
| `Tab` | Title ↔ content |
| `Ctrl+P` | Split ↔ edit |
| `Esc` | Back |

## Search / Tags / Folders

| Key | Action |
|-----|--------|
| type | Search or filter |
| `↑`/`↓` | Select |
| `Enter` | Open / apply filter |
| `Esc` | Back |

Mouse is supported throughout: click sidebar entries, list rows, and editor panes; scroll lists
with the wheel.
